Projects

Learner App

In order to help my Dutch learning journey, using SwiftUI and Firebase I created a Learner App. It allows users to register, add words to their personal dictionary and train the words by translation. I've also added Italian as one of the languages and a randomizing quizzing algorithm to ensure that words won't repeat too often.

Beer App

Using SwiftUI I created a beer app which allows the user to document the beers they have drunk in a clear way. Using Core Data I was able to make the app function even without internet connection. The app allowed to add custom pictures from by taking a photo or by selecting one from the photo library.

Restaurant App

I've always had an issue with picking restaurants when going out to the city, especially when I wasn't alone. Using React Native ,Node JS and Google Maps API I made an app that allowed users to connect and swipe through different restaurants. Once both users like the same restaurant the "game" ends.
